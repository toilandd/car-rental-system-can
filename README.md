# Car Rental System

A simple Java project developed for the OOP Final Exam.  
The system manages car rentals using basic OOP principles.

# Features
- Add cars  
- Rent a car  
- Return a car  
- List available cars  

# OOP Concepts Used
- Inheritance 
- Polymorphism   
- Interface 
- Encapsulation
